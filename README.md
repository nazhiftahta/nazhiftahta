## Featured Projects

### Public Discourse Intelligence: MBG Tweet Classification & Social Network Analysis

An end-to-end NLP and network analytics project for understanding Indonesian public discourse around MBG policy conversations.

**Technical scope**

- Trained and optimized **IndoBERT multi-class classifiers** on **5,000 labeled Indonesian tweets**
- Classified MBG-related public discourse into **8 policy-related categories**
- Built a complete NLP pipeline covering preprocessing, fine-tuning, evaluation, and inference on **1,500 unseen tweets**
- Optimized **balanced accuracy** to improve performance under class imbalance
- Conducted social network analysis on **15,000 tweet interactions** using **NetworkX** and **Gephi**
- Analyzed influence propagation, community clustering, polarization, and interaction dynamics

**Stack:** Python, IndoBERT, Hugging Face Transformers, Pandas, Scikit-learn, NetworkX, Gephi, NLP, Social Network Analysis

### AI-powered EduMetrix

A predictive academic performance and recommendation system for education analytics.

**Technical scope**

- Deployed the analytics interface using **Streamlit**
- Integrated **Ollama** and **Llama 3.2** for AI-generated academic recommendations

**Stack:** Python, Scikit-learn, Pandas, Streamlit, Ollama, Llama 3.2, Machine Learning
